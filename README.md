# DevWeb Simulado Quiz

Este repositório contém um **simulado interativo desenvolvido em PHP**, criado como atividade prática da disciplina **Desenvolvimento Web**. O projeto utiliza integração com **API de mensagens e tradução**, tornando a aplicação mais dinâmica e próxima de cenários reais.

## 🎯 Objetivo

* Praticar conceitos de desenvolvimento web com PHP
* Criar um sistema de simulado interativo
* Integrar APIs externas para enriquecer a aplicação
* Aplicar lógica de programação no back-end

## 📚 Conteúdos abordados

Durante o desenvolvimento, foram aplicados conceitos como:

* Estrutura de páginas com HTML
* Estilização com CSS
* Programação back-end com PHP
* Consumo de APIs externas
* Manipulação de dados e respostas do usuário
* Lógica de validação de respostas

## 🛠️ Tecnologias utilizadas

* PHP
* HTML5
* CSS3
* API de mensagens
* API de tradução

## ⚙️ Funcionalidades

* Exibição de perguntas em formato de quiz
* Seleção de respostas
* Validação de acertos e erros
* Exibição de pontuação final
* Navegação entre perguntas

## 💡 Observações

* Projeto desenvolvido para fins acadêmicos
* Focado em aprendizado e prática
* Pode ser expandido com novas perguntas, banco de dados ou integração com API
